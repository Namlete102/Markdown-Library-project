# 1. Mục lục - Table of contents

## 1.1 Cài đặt

Đối với người dùng Visual Studio Code, lựa chọn tốt nhất để tạo mục lục trong tài liệu Markdown là sử dụng tiện ích mở rộng (extension) là **Markdown All in One**.

Để cài đặt tiện ích mở rộng, hãy mở phần mềm Visual Studio Code, nhập tổ hợp phím `Ctrl` + `P`, nó sẽ hiện lên thanh tìm kiếm, dán lệnh dưới đây vào:

```
ext install yzhang.markdown-all-in-one
```

sau đó nhấn Enter để dẫn đến tiện ích mở rộng (extension) cần tải để sử dụng trong việc tạo mục lục.

Hoặc cũng có thể tải tiện ích mở rộng từ [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one) đây.

## 1.2. Sử dụng

Để tạo được mục lục, nhập tổ hợp phím `Ctril` + `Shift` + `P`, gõ và chọn trên thanh tìm kiếm dòng chữ `Markdown All in One: Create Table of Contents`.

Nếu có sửa đổi hoặc thêm các tiêu đề, làm tác động đến mục lục, người dùng chỉ cần nhập tổ hợp phím `Ctrl` + `S` (đây cũng là thao tác lưu tệp (save)) để hệ thống tự động cập nhật.
