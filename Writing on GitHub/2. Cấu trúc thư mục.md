# 2. Cấu trúc thư mục - Folder Structure

## 2.1. Cài đặt

1. Cài đặt tiện ích mở rộng sau [Draw Folder Structurethông](https://marketplace.visualstudio.com/items?itemName=jmkrivocapich.drawfolderstructure#:~:text=Flexible%20Exclusions:%20Exclude%20specific%20files,js%22) đây.

2. Thao tác tải về giống với việc chúng ta đã làm ở bài học [Mục lục](./1.%20Mục%20lục.md) với việc cài đặt tiện tích mở rộng **Markdown All in One** để tạo mục lục nhanh trong Markdown. 

## 2.2. Sử dụng 

Người dùng thực hiện các bước sau sau: 
1. Nhấp chuột phải vào thư mục mà người dùng mong muốn trong trình quản lý tệp Visual Studio Code.

2. Chọn **Generate Markdown Structure** từ menu ngữ cảnh.
3. Cấu trúc thư mục được định dạng bằng Markdown sẽ tự động được sao chép vào Clipboard của người dùng.
4. Hãy dán đoạn mã này vào nơi khác của tệp hoặc tài liệu mà người dùng cần.

Ví dụ cấu trúc thư mục **Writing on Github** của dự án Markdown-Library-project đây: 

````markdown
```
<!--chèn vào đây-->
```
````

Kết quả:

```markdown
<!--chèn vào đây-->
```
