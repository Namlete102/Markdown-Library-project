# 2. Cấu trúc thư mục - Folder Structure

## 2.1. Cài đặt

1. Cài đặt tiện ích mở rộng (extension) [Draw Folder Structurethông](https://marketplace.visualstudio.com/items?itemName=jmkrivocapich.drawfolderstructure#:~:text=Flexible%20Exclusions:%20Exclude%20specific%20files,js%22) đây.

## 2.2. Sử dụng 

Thực hiện các bước sau đâu: 
1. Nhấp chuột phải vào thư mục người dùng mong muốn trong trình quản lý tệp Visual Studio Code.
2. Chọn **Generate Markdown Structure** từ menu ngữ cảnh.
3. Cấu trúc thư mục được định dạng bằng Markdown sẽ tự động được sao chép vào Clipboard của người dùng.
4. Hãy dán đoạn mã này vào nơi khác của tệp hoặc tài liệu mà người dùng cần.

Ví dụ:

````markdown
```
<!--chèn vào lại-->
```
````

Kết quả:

```markdown
<!--chèn vào lại-->
```
